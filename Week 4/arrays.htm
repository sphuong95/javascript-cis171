Arrays

Assume that you live in an apartment building that contains 100 apartments. Does every apartment have a different mailing address? The answer is no, all the 100 apartments have the same mailing address, but they are distinguished from each other by apartment numbers. When you write JavaScript statements, lots of times you need to create variables that have more than one member (similar to the apartment building.) In JavaScript we call this object an array.  An array contains a set of data represented by a single variable name. You can think of an array as a collection of variables contained within a single variable. You use arrays when you want to store groups or lists of related information in a single, easily managed location. 

Arrays are represented in JavaScript by the Array object. The Array object contains a special constructor named Array() which is used for creating an array. You create new arrays by using the keyword new and the Array() constructor with the following syntax:

var variablename = new Array();

Within the parentheses of the Array() construction, you can include an integer number that represents the number of elements to be contained in the array. Each piece of data contained in an array is called an element. An index is an element'ss position with an array. The numbering of elements within an array starts with an index number of zero. Most programming languages require that all of the elements in an array be of the exact same data type. However, in JavaScript the values assigned to array elements can be of different data types (although this is not recommended).

Example:

var Apartment = new Array();

To refer to the members of the array you have to start with the array name followed by the index number enclosed in square brackets. For example to refer to the first member of the Apartment array you would type:

Apartment[0];

Note that the term Array must be capitalized.

Example: Create an array named Johnson. Assign the following names to the elements of the Johnson array: John, Kathy, Brian, Nicole.

<script type="text/javascript">
    var Johnson = new Array();
    Johnson[0]="John";
    Johnson[1]="Kathy";
    Johnson[2]="Brian";
    Johnson[3]="Nicole";
</script>
 

OR

<script type="text/javascript">
    var Johnson = new Array ("John","Kathy","Brian","Nicole");
    document.write("second member is: "+ Johnson[1]);
</script>

 

Practice: Create an array that assigns the days of the week to the array members. Name the array "Weekdays".

<script type="text/javascript">
    var Weekdays = new Array ("Sunday", "Monday", "Tuesday","Wednesday", "Thursday", "Friday", "Saturday");
</script>  

 

In JavaScript, some of the objects in the browser object model represent arrays. Some of those objects are frames, forms, links, and images.   For example. the images[ ] array represents all the images on a Web page. The first image is represented as images[0], the second image is represented as images[1], and so on.
