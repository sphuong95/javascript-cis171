Boolean Values and Variables

Boolean values are logical values which can be either true or false. To assign true or false values to a boolean variable, you can type the following statements:

var varname=true;
var varname=false;

 

Example:

 <script type="text/javascript">
 var signin=true;
  if (signin==true){
  alert("You have successfully signed in!");
  }
  else{
  alert("You have to sign-in inorder to proceed");
  }
   </script>

Note that since the boolean values of true and false are not text strings, and you should not enclose them in quotations.

You can modify the above if statement to:

if(signin)

and the result you'll get will be identical to the above example.

elseIf Statements

In some cases you may want the statements executed by the if statements to make other decisions. Sometimes, one decision-making statement is contained within another decision-making statement. In those cases, you can use the else if statements as shown below:

if (condition1) {
  Statements to be executed if
  condition1 is  true
}
else if (condition2) {
 Statements to be executed
 if condition2 is  true
}
...........
...........
}
else{
  Statements to be executed if none of the conditions are true
}

 

Example: Create a statement that determines the discount percentage based on the number if items purchased. If the customers buys less then 10 items, the discount is zero. If the customers buy 10 to less then 20 items, the discount is 10%. If the customers buy 20 to less than 50 items, the discount is 20%. If the customers buy 50 to less than 100 items, the discount is 25%. If the customers buy 100 items or more, then the discount is 30%. Display the discount in an alert box.

 

<script type="text/javascript">
var items;
var discount;
items=prompt("How many items do you want to purchase?");
if (items<10){
  discount=0;
}
else if (items<20){
  discount=.1;
}
else if (items<50){
  discount=.2;
}
else if(items<100){
  discount=.25;
}
else{
  discount=.3
}
alert("Your discount is: " + discount*100 +"%.");
</script>

Switch Statement

Another JavaScript statement that allows you to make a decision based on various choices is the switch statement. The following shows the syntax of the switch statement.

switch(expression)
{
case 1:
 execute code block 1;
 break; 
case 2:
 execute code block 2;
 break;
.............
.............
default:
 code to be executed if none of the cases are true
}

 

The switch statement evaluates an expression and executes a statement or a block of statements based on the value of the expression. The JavaScript interpreter checks each case against the value of the expression until a match is found. If nothing matches, a default condition will be used.

The way the switch statement works is that first, the value of the expression is compared with the values for each case in the structure. If there is a match, the block of code associated with that case is executed. The break statement instructs the interpreter to exit the switch statement after a match is found.

Example: Create a switch statement that determines the status of a high school student based on the number of years they have been in high school.

<script type="text/javascript">
  var year;
  var status;
  year=parseInt(prompt("How many years have been in high school?"));
  switch(year){
case 1:
  status="Freshman";
  break;
 case 2:
  status="Sophomore";
  break;
 case 3:
  status="Junior";
  break;
 case 4:
status="Senior";
  break;
}
  document.write("You are a " + status);
 </script>
