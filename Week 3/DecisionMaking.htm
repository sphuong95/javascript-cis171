if Statement

The if statement is one of the most common type of decision making statements. You use the if statement in JavaScript to make decisions. The ability to have a program make a decision and react accordingly is a key to writing flexible programs that can easily handle a range of scenarios.  The if statement is used to execute specific programming codes if the evaluation of a conditional expression returns a true value. The syntax for a simple if statement is as follows:

if(conditional expression){
       statements;
}

Example:

if (age>=18){
   document.write("You are an adult");
}


Note: Remember that Javascript statements are case sensitive, so if you change the capitalization of the if statement  you will get an error message.

 


The if statement contains three parts: the keyword if, a conditional expression enclosed within parentheses, and executable statements. If the condition being evaluated returns a value of true, then the statement immediately following the conditional expression executes.

 


After an if  statement executes, any subsequent codes execute normally. You can use a command block to construct a decision-making structure using multiple if statements. A command block is a set of statements contained within a set of braces. Each command block must have an opening brace ( { ) and a closing brace ( } ). When an  if statement contains a command block, the statements in the command block execute when the if statement condition evaluates to true. When the condition evaluates to false, the command block is skipped, and the statements that follow execute.

 


Example:

 

if (quantity>=100) {
    discount=0.1;
    document.write("Your discount is: " +  discount*100 + "%.");       
}

Comparison Operators

If else statements

In some situations you may want to execute one set of statements when the condition evaluates to false, and another set of statements when the condition evaluates to true. In that case, you need to add an else clause to your if statement. You can use command blocks to construct an if...else  statement as follows:

if (conditional expression) {
statements;
}
else {
statements
}

Example

if(age>=18){
status="Adult";
}
else {
status="Minor";
}
