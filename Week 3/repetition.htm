Loops

A loop statement is a control structure that repeatedly executes a statement or a series of statements while a specific condition is true or until a specific condition becomes true. JavaScript contains various types of loops. In this lecture, you are going to learn about the most frequently used loops which are the while and the for loops.

while

One of the simplest types of loop statements is the while statement, which repeats a statement or series of statements as long as a given conditional is to true. The syntax for the while statement is as follows:

while (conditional expression) {
      statement(s);
}

The conditional expression in the while statement is enclosed within parentheses following the keyword while. Each repetition of a looping statement is called an iteration. When the conditional expression evaluates to false, the loop ends and the next statement following the while statement executes.

You track the progress of a while statement, or any other loop, with a counter. A counter is a variable that increments or decrements with each iteration of a loop statement. To ensure that the while statement will eventually end, you must include code within the body of the while statement that changes the value of the conditional expression. In an infinite loop, a loop statement never ends because its conditional expression is never false.

Example:


<script type="text/javascript">   
    var result=1;   
    while (result <=50) {       
        document.write(result + "<br />");       
        result=result*2;   
    }    
</script>

Result

 


Note that the statements document.write and result=result*2 will execute until result becomes more than 50.

Practice: Create a while loop that asks for a number and displays the square root of the number. The program should repeat this until the visitor enters zero as the number.

 

Answer:

 

<script type="text/javascript">
    var num;
    num=prompt("Enter a number:");
    while(num!=0) {
        document.write(num*num +"<br />");
        num=prompt("Enter a number:");
    }
 </script>

for Statement

The for statement is used for repeating a statement or series of statements as long as a given conditional expression is true. The for statement performs essentially the same function as the while statement: if a conditional expression within the for statement evaluates to true, then the for statement executes and will continue to execute repeatedly until the conditional expression evaluates to false.

One of the primary differences between the while statement and the for statement is that in addition to a conditional expression, the for statement can also include the code that initializes a counter and changes its value with each iteration. This is useful because it provides a specific place for you to declare and initialize a counter, and to update its value, which helps prevent infinite loops.

The syntax of the for statement is as follows:

for (counter initialization; test condition; increment or decrement the counter) {
          statement(s); }

Example:


<script type="text/javascript">
    var result;
    for (result =1; result<=50; result=result*2) {
     document.write(result + "<br >");
    }
</script>

Practice: Create a Web page that shows numbers 1 through 25, and their squares in a table.

 

Answer:

 

<script type="text/javascript">
     var num;
     document.write
     ("<table border='1' style='width:30%'>");
     for (num=1; num<=25; num++){
           document.write(
           "<tr> <td>"+num + "</td> <td>" +
           num*num  +  "</td></tr>");
     }
     document.write ("</table>");
 </script>

 

Result:

Note that you should only include the statements that should repeat within the body of the loop.
