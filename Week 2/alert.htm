Dialog Boxes

Alert

At times it might be necessary to notify the user of possible issues or problems with their input. JavaScript provides two methods to do this. These methods are the alert() and the confirm() methods, and both are very useful for getting the user's attention. The only difference between the two is that the confirm() method returns the user's response, while alert displays a message. Depending on the situation, the alert() method may be all that is required. The alert method is an integral part of letting the user know of errors found in their input.

The syntax of the alert method is as follows:

alert("Message");

Example: Create an alert box that displays the message "Welcome to My Web Page".

<script  type="text/javascript">
       alert("Welcome to My Web Page");
</script>

Confirm

The confirm method displays a message box just like the alert method, but also includes a Cancel button. If the visitor clicks the OK button, the true value is returned. If the visitor clicks the Cancel button, the false value is returned.  The following displays the syntax of the confirm method:

confirm("Message");

Example: Create a dialogue box that contains both OK and Cancel buttons.  If the user clicks the OK button, a message will display that says "I will continue". If the user clicks the Cancel button, a message will display that says "I will quit now".

<script  type="text/javascript">
   var answer=confirm("Do you want to continue?");
   if (answer==true){
    document.write("I will continue.");
   }
else {
    document.write("I will quit now.");
  }
</script>

Prompt

The prompt method collects a piece of data from the user and assigns it to a variable. The following displays the syntax of the prompt method:

var VariableName = prompt("Prompt", "Default Value");

The default value is optional, but if you decide to add a default value, the value will display in the text box that is provided in the prompt's dialogue box.

Example:  Create a prompt box that asks for the user's first name and then assigns the name to a variable named FirstName.

<script  type="text/javascript">
     var FirstName=prompt("Enter your first name", "First Name");
</script>

 

Now assume that you want to take the user's first name and display a user friendly message. The following example creates a message using the visitor's first name.

<script  type="text/javascript" >
    var FirstName=prompt("Enter your first name", "First Name");
    document.write("<h1> Hi " + FirstName + ". Welcome to My Home Page. </h1>");
</script>
