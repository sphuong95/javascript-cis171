Variables

The values a program stores in computer memory are commonly called variables. A variable is a placeholder used for storing the data in a program. Data stored in a specific variable often changes. To use a variable in a program, you first have to write a statement that creates the variable and assigns it a name (declaring a variable.) In addition to variables, users can create constants. A constant contains information that does not change during the course of program execution.

A variable  is a reserved area of memory where a value can be stored and retrieved while a program/script is running. When a program/script is loaded into memory, the bytes fall into one of two categories: instructions and data. Instructions tell a program/script what to do. Data is used by the instructions. Some data is constant, meaning it cannot be changed while the program/script is running. This is true of any literal numbers or strings coded into the script. Some data, however, may not be known at the time the script begins or may need to change while the script is running. This is true of values which are calculated, manipulated, extracted from forms or cookies, or otherwise input to the script. Since these values can change, they are known as variables.

Variable names

When declaring a variable, it is wise to follow generally accepted naming conventions. Chief among these is Hungarian notation. When a variable consists of two or more words such as "checking" and "account", Hungarian notation says to put the words together without any separators such as the underscore between them, and then capitalize the first letter of the second word and beyond. Using the words given above, the variable name becomes checkingAccount. This makes the variable name easier to read. The first letter is left lowercase to help distinguish the variable from more complex objects that will come later. Hungarian notation is also used for function names in JavaScript.

When naming a variable, programmers need to adhere to the following rules: 

  1. Variable names must begin with an uppercase or lowercase ASCII letter, dollar sign ($), or underscore ( _ ).
  2. You can use numbers in a variable name, but not as the first character.
  3. You cannot include spaces in a variable name.
  4. You cannot use reserved words for variable names.

Developers cannot use the JavaScript Keywords to create variables. Some programmers tend to capitalize the first letter or join reserved words with underscores to create variables. This approach is NOT recommended because it confuses other developers.

Note: Variable names are case sensitive.  If you receive errors that a variable is undefined, check to ensure that the capitalization used in a specific statement is the same as when the variable was declared.

Declaring Variables

In JavaScript, you usually use the reserved keyword var to create variables. For instance, to create a variable named  myVariable, you use this statement:

 var myVariable;


There are two steps to create a variable: 
  1. Declare the variable (as shown above)
  2. Initialize the variable

The '=' sign is used to initialize the variable, such as: 

            varvariable_name = value;

Example:  

<script type="text/javascript">
            
    var age = 21;
    var firstname = "John"
</script>


 

The value you assign to a variable can be a literal string or a numeric value. When you assign a literal string value to a variable, you must enclose the text in quotation marks. However, when you assign a numeric value to a variable, do not enclose the value in quotation marks or JavaScript will treat the value as a string instead of a number.

The '+' sign can be used to join strings together and for arithmetic operations. This is called String concatenation.

Example: Create two variables by the names firstname and lastname and assign your first and last names to those variables.  Using the document.write method, display the full name.

<script   type="text/javascript">
    
     var firstname = "Sally ";
     var lastname = "Anderson";
     document.write(firstname + lastname);
   </script>


 

Result:

SallyAnderson

Example:  Create two variables by the names message1 and lastname.  Assign a message to message1 and your last name to lastname.  Display the massage and your last name on the screen.

<script type="text/javascript">
 
     var message1 = "Hello";
     var lastname = "Anderson";
     document.write("<h1>"  + message1 + " " + lastname + "</h1>");
</script>

Result:  

Hello Anderson

 

Note: The empty double quotations in the above example places a space between the words Hello and Anderson.  What happens if you removed them from the example?

You can change the variable's value at any point in a script by using a statement that includes the variable's name, followed by an equal sign, followed by the value you want to assign to the variable. For example you can change the value of the First and Last names in the above example. 

Assume that  you have declared a city variable  with a value of "Las Vegas", a state variable with a value of "NV", and a zip variable with a value of "89119", you might want to concatenate them into a single string variable named citystring.  The statement citystring = city+state+zip; would return "Las VegasNV89119".  In order to get the desired result, your statement would need to read:

citystring = city + ", " + state + " " + zip;.  

Note: Variable names are case sensitive.  If you receive errors that a variable is undefined, check to ensure that the capitalization used in a specific statement is the same as when the variable was declared.

prompt Method()

The prompt() method collects a piece of data from the user and assigns it to a variable. The following displays the syntax of the prompt method:

var Variablename = prompt("Prompt", "Default Value");

The default value is optional, but if you decided to add a default value, the value will display in the text box automatically. The prompt() method is used to give the user a dialog box with a text field.  In order to use that response in your program, you need to assign its value to a variable.  

 

Example:  Create a prompt box that asks for the user's first name and then assigns the name to a variable named firstname.

 

<script type="text/javascript">
   
     var firstname=prompt("Enter your first name", "First name");
</script>

Click here to see the result

Now assume that you want to take the user's first name and display a user friendly message. The following example creates a message using the visitor's first name.

<script type="text/javascript">
    var firstname=prompt("Enter your first name", "First name");
    document.write("<h1> Hi " + firstname +
    ". Welcome to My Home Page. </h1>");
</script>

parseInt() Method

Assume that you want to retrieve two values from the users which are declared as number1 and number2. The total variable is supposed to add up the two numbers and display it on the screen.

<script   type="text/javascript">
    var  number1=prompt("Enter the first number", "Number 1");
    var  number2=prompt("Enter the second number", "Number 2");
    var total=number1+number2;
    document.write(number1 + "+" + number2 + "=" +  total);
</script> 

Click here to see the result.  

Type any two numbers like 10 and 20 in the two textboxes. Does the result surprise you? 

 

In this case, JavaScript does not know whether to concatenate or add the values.  By default, JavaScript interprets any results of prompt() dialog boxes as strings.  As a result, it will treat the two numbers as strings and concatenate them into a single string. So if you have typed 10 and 20 in the two textboxes, the result would be 1020 instead of 30.

The parseInt() method addresses the problem described above by instructing the JavaScript interpreter to convert the value into an integer.  An integer is a whole number without decimal points. As you can see in the following statements, JavaScript knows that the values are numbers and that it should treat the plus sign as an addition operator, not a concatenation operator.

<script type="text/javascript">
        var number1=prompt("Enter the first number", "Number 1");
        number1=parseInt(number1);
        // converts the string into an integer value
        var number2=prompt("Enter the second number", "Number 2");
        number2=parseInt(number2);
        var total=number1+number2;
        document.write(number1 + "+" +number2 + "=" + total);
</script>
 

Click here to see the result.

Now, try to insert two whole numbers (numbers with decimal places) in the above textboxes. Does the total display the decimals? The answer is no, since the parseInt() method just converts the strings into whole numbers. If your numbers contain decimal places, you can use the parseFloat() method instead.
